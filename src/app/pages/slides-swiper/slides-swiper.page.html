Se usa Swiper y no Slides

En v6 de Ionic el ion-slides se estableció como deprecates y en v7 se retiró definitivamente. Por tanto ignoramos el tutorial para este ejercicio y usaremos el swiper recomendado.

Pasos instalacion

Actualizar Ionic:

npm install @ionic/angular@latest

Instalar la dependencia de Swiper:

npm install swiper@latest

En caso de no usar app.module.ts en v8, cambiar app.component.ts:

import { CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';

Añadir el siguiente código para registrar Swiper:

import { register } from 'swiper/element/bundle';

Agregar el esquema:

schemas: [CUSTOM_ELEMENTS_SCHEMA]
